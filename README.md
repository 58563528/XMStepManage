# XMStepManage

### 测试机器信息
1. 手机型号：小米6X
2. MIUI版本：MIUI 10.2.1.0
3. Android版本：8.1.0
4. 是否ROOT：是
5. 是否Xposed：是(Edxposed 90.0-v0.4.6.0_beta(4471)(YAHFA))
6. 是否核心破解：是
7. 是否第三方ROM：是(mandfx)
