# XMStepManage
对小米系统的步数进行管理，在MIUI11（Android9-10）可能存在BUG(具体原因为权限不足)。当然也可能是需要ROOT和核心破解...

### 测试机器1信息
1. 手机型号：小米6X
2. MIUI版本：MIUI 10.2.1.0
3. Android版本：8.1.0
4. 是否ROOT：是
5. 是否Xposed：是(Edxposed 90.0-v0.4.6.0_beta(4471)(YAHFA))
6. 是否核心破解：是
7. 是否第三方ROM：是(mandfx)
### 测试机器2信息-------
1. 手机型号：红米5Plus
2. MIUI版本：MIUI 10.1.1.0
3. Android版本：7.1.2
4. 是否ROOT：是
5. 是否Xposed：是(xposed V89)
6. 是否核心破解：是
7. 是否第三方ROM：是(mandfx)
